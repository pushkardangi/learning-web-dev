Notes For JavaScript

What can In-Browser JS do ?

Add new html and change existing html from DOM
React to events (action performed by user) (response from server) (mouse movement) Ajax request Get and set cookies and use local storage

What can't in-browser JS do ?

Read/Write to and from computer hard disk.
Works on same origin policy : restrict a website to interact with another website (having different origins)
(but it can also access the permitted resources)

What makes JS a strong language

Html / CSS support Simple API -> fetch data Major modern browsers(enabled by default)
