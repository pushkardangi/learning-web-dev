Types of Css :

  1. inline css_writing css in the tags itself
  2. internal css_writing css in head and using it through class and id or by simply mentioning tags
  3. external css_making css file in other doc and linkig it to markup

Rules you may remember for easy handling :

  1. property which was given in the latest time overwrites the property which was given earlier for the same text
  2. link css file with <link> tag in head tag
  3. if you are giving internal css, they are also written in head tag
  4. Html will give more preference to inline css than internal or external css

SELECTORS IN CSS #15

What is a slector

  1. css selectors are used to find the element whose proprty will be set
  2. selectors are used to target the html element
  3. selectors makes it easy for us to easily target single/multiple html elements in the markup

Types of basic css selectors

  • css element selector
  • css id selector
  • css class selector
  • css grouping selector

CSS Syntax

h1{color: blue; font-size:12px;}         [selector: h1] [property: color,font-size] [value: blue, 12px]
