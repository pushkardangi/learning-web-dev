What can In-Browser JS do ?

  1. Add new html and change existing html from DOM
  2. React to events (action performed by user) (response from server) (mouse movement)
  3. Ajax request
  4. Get and set cookies and use local storage

What can't in-browser JS do ?

  1. Read/Write to and from computer hard disk.
  2. Works on same origin policy : restrict a website to interact with another website (having different origins)
    (but it can also access the permitted resources)

What makes JS a strong language

  1. Html / CSS support
  2. Simple API -> fetch data
  3. Major modern browsers(enabled by default)
